# Robust Airborne Fraction Trend Estimation

Repository for the paper "Multi-source land-use emissions reveal rising airborne fraction" by J. Eduardo Vera-Valdes. This repository contains the code and analysis on estimating the airborne fraction using multilple realizations of the measurements to obtain robust estimates of the airborne fraction and its uncertainty. 

The code is written in Julia and R, and the data is available in the `data` folder. The main scripts for the analysis are in the `scripts` folder. The functions used in the analysis are in the `estimation_functions.jl` file. The results of the analysis are saved in the `results` folder. The paper is available in the `_manuscript` folder and the figures are saved in the `figures` folder.
