# Robust-Airborne-Fraction

Repository for the paper and analysis on estimating the airborne fraction using multilple realizations of the measurements to obtain robust estimates of the airborne fraction and its uncertainty. The code is written in Julia and the data is available in the `data` folder. The main script for the analysis is `af_2step_all_lulc_measurements.qmd` and the functions used in the analysis are in the `estimation_functions.jl` file. The results of the analysis are saved in the `results` folder. The paper is available in the `_manuscript` folder and the figures are saved in the `figures` folder.
